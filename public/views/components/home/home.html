My Angular Bootstrap Web App

Welcome to a super easy, pre built web app using angular and bootstrap which will allow you to start creating content right now. No need to get the basics set up, just start putting your specific content into templates, join them to a route, create some controllers etc. and your on your way!

Please note: This is a developer pack, intended for use by developers.

This is intended to cover the first 10% of setting up a new project. Once you have this pack installed, the work actually starts. I have left out most stuff because there are soooo many ways to complete the required features of any given web app. This is just the most typical stuff you will need on the front end to get going.

What's in the box?

Core Components
  • AngularJS 1.5.6 (https://angularjs.org/)
  • Bootstrap 3.3.6 (http://bootstrapdocs.com/v3.3.6/docs/)
  • Angular UI Bootstrap (https://angular-ui.github.io/bootstrap/)
  • Angular UI Router (http://angular-ui.github.io/ui-router/site/#/api/ui.router)
  • UI Router Meta-Tags (https://github.com/tinusn/ui-router-metatags)
  • Other Angular Components
  • jQuery 2.2.4 (https://jquery.com/)
Some Fun Stuff and Utils
  • MomentJS (http://momentjs.com/)
  • Animate.css (http://daneden.github.io/animate.css/)
  • Font Awesome Icons (http://fontawesome.io/icons/)
  • VidoeJS + Youtube Plugin (http://videojs.com/)(https://github.com/videojs/videojs-youtube)
  • LightGallery (http://sachinchoolur.github.io/lightGallery/)
  • Please Wait Splash Screen (http://pathgather.github.io/please-wait/)
  • Full Suite of Source Sans Pro Font Pack (http://adobe-fonts.github.io/source-sans-pro/)
Custom Web App Components
  • Set of LESS files with things to help you build web application
  • Defined Angular Folder Structure which might serve you well as your web application grows
  • .htaccess file for deploying the front-end on an Apache web server
  • Grunt build automation with watch task to aid development work flow
  • A couple of useful Angular Directives
  • A NodeJS backend for rapid development

So why do this? There are heaps of other starter templates/seed templates/starter packs for Angular and Bootstrap, why not just use one of those? Answer is, you can!
